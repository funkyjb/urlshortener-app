Welcome to the URL shortener app

Since you were able to get here, the application works correctly. Anyway let's introduce the steps needed for running the app.

Prerequisites

Following needs to be installed on your machine:

  • Java 11 or higher
  • Git

Run

This guide describe steps to run the application on your local machine.
  • clone the repository with Git from remote location
  • find the .jar file inside your local cloned project in directory /package
  • run the .jar file using java -jar <filename>.jar

After that the application is ran in your JVM.

Usage

The application is accesible through a browser on address localhost:8080. In order to get to this page you used localhost:8080/help

You can use the application throug this API:

POST /account

Creates an account with requested account ID and returns password when account with the ID doesn't exist.

Request:

                    
                    {
                        "accountId" : "{yourAccountID}"
                    }

Response:

                    
                    {
                        "success": true,
                        "description": "Account has been successfully created",
                        "password": "7JQ0UYmX"
                    }
Account ID and returned password should be used as basic authorization credentials.

POST /register

Registers URL and returns shorten version. Requires basic authorization.

Request:

                    
                    {
                        "URL": "https://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html",
                        "redirectType": 301
                    }
redirectType is optional field, no need to specify.

Response:

                    
                    {
                        "shortUrl": "http://localhost:8080/Sq9x1h"
                    }

GET /statistic/<accountId>

Returns statistic of url calls for specific user. Requires basic authorization.

Response:

                    
                    [
                        {
                            "https://www.w3.org/Protocols/rfc2616/rfc2616-sec10.html": 2
                        },
                        {
                            "https://wst.tv": 8
                        },
                        {
                            "https://www.nhl.com/": 3
                        }
                    ]

GET /<urlId>

Redirects you to the registered url. Simply GET an shortUrl returned in /register call. Requires basic authorization.

Response:

The page content of registered URL.

GET /help

This help page.

GET /h2-console

Interface for accessing the in-memory database.
